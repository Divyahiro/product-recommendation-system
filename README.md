# Product Recommendation System

A basic product recommendation system that suggests products using user ratings and cosine similarity.

## Status
Completed (Version 1)

## Objective
Build a recommendation system that recommends products based on user ratings and similarity between users.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Workflow
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- User-Product Matrix Creation
- User Similarity Calculation (Cosine Similarity)
- Product Recommendation
- Save Recommendations

## Repository Structure
```
data/
notebooks/
models/
src/
README.md
requirements.txt
```

## Dataset
Amazon Product Reviews Dataset

## Results
- Built a user-product interaction matrix.
- Computed user similarity using cosine similarity.
- Generated product recommendations for a sample user.
- Saved the recommended products as a CSV file.

## Future Improvements
- Item-based collaborative filtering.
- Matrix factorization (SVD).
- Hybrid recommendation system.
- Deploy using Streamlit or Flask.
